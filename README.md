# Gamify Todo

## Overview
A mobile app that turns task management into a game. Users complete daily tasks to earn points and purchase rewards from an in-app store.

## Key Features
- **Task Management**: Create, edit, and track tasks
- **Routines**: Set up recurring tasks  
- **Reward System**: Earn points for completed tasks and spend them in the store
- **User Profile**: Track stats and progress
- **Multi-language Support**: English and Turkish
- **Notifications**: Task reminders and alerts
- **Traits System**: Develop and level up character traits

## Tech Stack
- **Flutter**: Cross-platform mobile framework
- **Provider**: State management
- **Hive**: Local database
- **Easy Localization**: Multi-language support
- **Firebase**: Backend & notifications

## Project Structure
```
lib/
├── Core/          # Helpers & utilities
├── Enum/          # App enums
├── General/       # Global config (colors, themes)
├── Model/         # Data models
├── Page/          # UI pages
├── Provider/      # State management
├── Service/       # Business logic services
└── main.dart      # Entry point
```
